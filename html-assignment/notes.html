HTML Tags and Description

Head
A container for metadata (title, links, scripts, etc.) about the HTML document.
Header
The header HTML element represents introductory content, typically a group of introductory or navigational aids.
Section
Defines a section in a document
Nav
Represents a navigation menu.
Body
Contains the content of the HTML document such as text, images, links, etc.
